# RuLibrary

## [RuLibraryとは]
discordRaiderを超簡単に作るために制作した荒らし用ライブラリーです。

## [特徴]
- Multi Raid対応✔
- 非同期処理で高速送信✔
- random String対応✔

## [インストール]
GitHubからクローンしてinstallしてください

```bash
git clone https://github.com/rur1jp/RuLibrary.git
cd RuLibrary
pip install -r requirements.txt
```

## [使い方]
- `Example`にコードがあります。それを見てください
